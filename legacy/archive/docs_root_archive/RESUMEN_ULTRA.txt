¿QUÉ ES?
════════════════════════════════════════════════════════════════════════════

Trading Phantom es un bot de trading automático para MetaTrader 5 que:
- Genera señales usando indicadores técnicos (EMA+MACD+RSI)
- Usa Machine Learning (Random Forest, 60% accuracy)
- Gestiona riesgo automáticamente (SL/TP)
- Almacena historial en SQLite (200+ trades)
- Expone API web con dashboard (Flask, puerto 5000)
- Contiene TODA la documentación necesaria (50+ archivos, 150+ KB)

Está 100% completo, auditado, documentado y LISTO PARA OPERAR.
